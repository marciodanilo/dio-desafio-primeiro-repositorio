# Desafio de projeto Git/Github

Repositório criado para o desafio de projeto da DIO.

## Links uteis:
[Sintaxe básica Markdown](https://www.markdownguide.org/getting-started/)

O principal objetivo do desafio é explorar a prática e uso da iteração entre o GIT e o GITHUB, desde como criar o repositório no GITHUB, fazer o download do projeto utilizando o GIT da sua maquina local, e também o caminho inverso, alterando os arquivos localmente e fazendo o upload para o GITHUB utilizando o GIT.
